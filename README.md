# 前端动效游乐场 (Frontend Effects Playground)

这是一个旨在展示现代 Web 前端动效、交互组件和视觉效果的项目。项目采用了独特的 **"Magazine" (杂志风格)** 设计语言，强调高对比度、硬阴影、粗边框和大胆的色彩搭配。

## 🌟 特色功能

-   **杂志风格 UI**: 采用高对比度的配色方案（深蓝、粉色、荧光黄），搭配硬阴影（Hard Shadows）和粗边框（Solid Borders），视觉冲击力强。
-   **动效展示区**: 包含基础 CSS 动画、高级动画、3D 变换等多种效果。
-   **交互组件库**: 包含按钮、卡片、模态框、通知（Toasts）、侧边栏等常用 UI 组件的深度定制。
-   **参数实时调节**: 配备属性控制面板，可实时调整动效参数并预览。
-   **代码预览与复制**: 完整展示 HTML/CSS/JS 代码，支持一键高亮复制。
-   **多主题支持**: 内置多套配色方案，并支持深色模式。
-   **响应式设计**: 完美适配移动端、平板和桌面端。

## 🛠️ 技术栈

-   **核心**: HTML5, CSS3 (Vanilla CSS), Modern JavaScript (ES6+)
-   **图标**: Font Awesome 6
-   **高亮**: Prism.js
-   **字体**: Inter (无衬线) & Merriweather (衬线)

## 📁 项目结构

-   `index.html`: 主入口文件
-   `css/`: 样式表目录
    -   `main.css`: 核心框架、布局与杂志风格基础
    -   `themes.css`: 主题变量定义（支持多主题切换）
    -   `animations.css`: 各种动画效果定义
    -   `components.css`: UI 组件样式
    -   `effects.css`: 视觉特技（玻璃态、新拟态等）
-   `js/`: 脚本目录
    -   `app.js`: 核心应用逻辑
    -   `state.js`: 状态管理器
    -   `utils.js`: 通用工具函数

## 🚀 快速开始

1.  克隆或下载本项目。
2.  直接在浏览器中打开 `index.html` 即可运行。
3.  通过侧边栏选择不同的分类，点击卡片预览效果。

## 🎨 设计哲学 - Magazine Style

本项目摒弃了平庸的渐变和模糊投影，采用了更加现代且具有个性的 **Magazine** 风格：
-   **Offset Shadows**: 使用偏移的实心阴影而非模糊投影，增加物体的物理感。
-   **Bold Outlines**: 统一的粗边框定义，使 UI 元素更加分明。
-   **High Saturation Palette**: 使用高饱和度的点缀色来引导用户的注意力。
